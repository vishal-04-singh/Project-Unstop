## üöÄ Quick Start

```bash
# Clone the repository
git clone https://github.com/Avenster/clinikally-project.git

# Navigate to project directory
cd clinikally-project

# Fresh installation
rm -rf node_modules
rm package-lock.json
npm install

# Start the application
npm run ios     # For iOS
npm run android # For Android
```
